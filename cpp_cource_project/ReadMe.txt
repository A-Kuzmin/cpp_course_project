Исходные данные 
Дано:
•	4 одномерных массива целых чисел, размерностями N1, N2, N3, N4. Первые два считываются из файлов, третий – генерируется случайными числами, четвертый – вводится с клавиатуры пользователем (диапазон допустимых значений см. в столбце 2).
•	Для первого и второго массивов выполнить задания, согласно варианту (см. столбец 3) 
•	Второй и третий массив сортируются, согласно варианту (см. столбец 4)
•	Третий и четвертый массив преобразовать по заданию (см. столбец 5) 
•	Исходные массивы и результат вывести на экран и в файл.

№	3.
Диапазон значений: [0, 5]

Задания для 1 и 2 массивов:
		1 – заменить на 0 все элементы между мин. и макс.
		2 – посчитать количество нулей
		
Направление сортировки:
		2 - по убыванию
		3 - по возрастанию
		
Задание для 3 и 4 массивов:
		3 – вставить макс. элемент после каждого 0
		4 – удалить все элементы между мин. и макс. элементами
